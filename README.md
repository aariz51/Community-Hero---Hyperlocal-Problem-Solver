# Community Hero — Hyperlocal Problem Solver (CivicPulse)

AI-powered civic issue reporting platform for the Vibe2Ship hackathon.
Citizens report local issues (potholes, leaks, streetlights) → Gemini AI
categorizes & scores → geo-mapped → routed & tracked to resolution.

## Stack
- React + Vite (plain, lean — AI Studio compatible)
- Firebase (Auth + Firestore) — planned
- Gemini API (Vision) + Google Maps Platform — planned
- Deploy: Google Cloud Run

## Dev
```bash
npm install
npm run dev   # http://localhost:5173
```
